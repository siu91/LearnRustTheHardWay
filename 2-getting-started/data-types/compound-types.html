复合类型

复合类型（Compound types）可以将多个值组合成一个类型。Rust 有两个原生的复合类型：元组（tuple）和数组（array）。

元组类型（tuple）

元组是一个将多个其他类型的值组合进一个复合类型的主要方式。元组长度固定：一旦声明，其长度不会增大或缩小。

元组中的每一个位置都有一个类型，而且这些不同值的类型也不必是相同的：

fn tuple_type() {
    let tup: (i32, f32, u8) = (32, 32.1, 32);
    // 使用"模式匹配"来"解构"元组
    let (x, y, z) = tup;
    println!("y 的值：{}", y)
}

fn main() {
 tuple_type();
}
